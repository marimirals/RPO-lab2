# Transport Card Payment Authorization System

REST API сервер для авторизации платежей транспортными картами с веб-интерфейсом.

## Технологии

### Backend
- **Go 1.21+** — язык программирования
- **Gin** — веб-фреймворк
- **SQLite3** — база данных
- **Goose** — миграции БД
- **JWT** — аутентификация
- **Swagger** — документация API
- **Nginx** — reverse proxy + HTTPS
- **Docker** — контейнеризация

### Frontend
- **React 18** — UI библиотека
- **Vite** — сборщик
- **React Router** — роутинг
- **Axios** — HTTP клиент

## Возможности

### Для пользователей
- Регистрация и авторизация по JWT
- CRUD транспортных карт MIFARE
- CRUD терминалов
- Просмотр транзакций
- Управление профилем

### Для администраторов
- Управление ключами шифрования
- Управление пользователями
- Полный доступ ко всем данным

### API для терминалов
- Авторизация платежных транзакций
- Загрузка ключей шифрования
